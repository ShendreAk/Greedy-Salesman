About Greedy Salesman

Greedy Salesman is a web app which uses time series forecasting model at the backend. The purpose of developing this app is to forecast furniture sales for the next upcoming month based on the input timestamp. This model is based on the famous superstore sales dataset from tableau. The codes for this project can be checked in my Github repo.

Dashboard

This dashboard is made and hosted on Tableau Public Tableau. This interactive dashboard helps us to understand the current year (2017) sales compared to previous year (2016) using key performance indicators. Additionally the dashboard represents country (overall)/region/state wise analysis of current year sales, profit, orders and quantity. All in all the dynamic dashboard helps in detailed analysis of current year performace of furniture sales to strategize effectively for upcoming months or years.
